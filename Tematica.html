Tematica de la App

La App tendra una interfaz un poco parecida a la aplicación conocida como: Indriver, tendra que registrarse con sus datos conrespondientes para crear una cuenta, los cuales son:

  • Nombre Completo.
  • Correo Electronico.
  • Número de Telefono.
  • Contraseña.
  • Datos de la Mascota.
    • Raza.
    • Nombre.
    • Edad.
    • Enfermedades(En caso de tener)

Luego de esto, la App llevara al usuario directamente a la pagina de inicio que tendra una interfaz de mapa, y alli podra solicitar finalmente su servicio. Para solicitar un servicio debera llenar los datos que piden en el recuadro que se encuentra en la parte inferior y especificar sus nesecidades para mayor efectividad, los datos para solicitar el servicio son:

  • Lugar de Encuentro.
  • Tipo de Mascota.
  • Tiempo de Duración.
  • Elementos de Distracción o Entretenimiento(en caso de no tener, la App tiene el servicio de rentar algunos elementos).
  • Metodo de Pago
    • Transferencia.
    • Efectivo.

Ya acordado el servicio, la App le ofrece al usuario el contacto directo del "Encargado" para mayor confianza, aparte de esto tambien el usuario puede ver en el mapa la posición actual de el "Encargado" y la mascota, esto mediante un chip de rastreo que incluye la aplicación.

El proyecto tambien contara con un sistema que tendrá una base de datos de mascotas o razas para conocer sus cuidados necesarios y el "peligro" de cada mascota y asi evitar tragedias.
